About

Hey, This is a Demo Project.Which I created using Angular, in which you can change your string into Upper case,lower case, remove white space and also clear text using button.The best thing about the project is that it shows converted data in different textarea. So you can easily perform different task in single string.However project uses very basic functionality so easy to learn but it is very useful and different from other available project in google.

History

How I get idea about this project? After learning angular, I was checking different available project but all of them are almost similar. So I choosed Pipeline concept in angular to create my project. Adding normal normal things like login, navbar it looks so beautiful.
